# Credentials Needed

Send these via secure channel (1Password/Bitwarden/ProtonMail). Do not send passwords in plain email.

## Access Checklist
- [ ] GTM Container: publish access for [your email]
- [ ] GA4 Property: Admin + Editor access
- [ ] HubSpot: Private App token or OAuth invite
- [ ] LinkedIn Ads: Campaign Manager access (at least Viewer)
- [ ] Instantly: Login + webhook/API token
- [ ] Google Drive folder for documentation delivery

## Optional (Advanced)
- [ ] BigQuery project access (for GA4 export)
- [ ] Cloud billing enabled for BigQuery
- [ ] Service account JSON for scheduled queries

## Secure Delivery Options
- Encrypted email (ProtonMail)
- Password manager share (1Password/Bitwarden)
- Secure file link with expiry (Drive/Dropbox)
