# Looker Studio Validation Checklist

- [ ] Each data source connected (GA4, HubSpot, LinkedIn, Instantly)
- [ ] Date controls default to Last 30 Days
- [ ] Filters respond correctly (Channel, Campaign)
- [ ] Calculated fields show correct formulas (variance, conversion rate)
- [ ] Currency formatting correct across charts
- [ ] Last refresh timestamp visible on cover page
- [ ] Export to PDF/CSV works without errors
- [ ] Stakeholder access verified (Viewer vs Editor)
