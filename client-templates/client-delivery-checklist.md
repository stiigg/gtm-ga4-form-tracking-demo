# Client Delivery Checklist

Run through this checklist before marking a project complete.

## Validation
- [ ] Reconciliation workbook updated with final variance notes
- [ ] GA4 DebugView shows all key events (form + ecommerce if applicable)
- [ ] Looker Studio dashboard refreshed and filters tested
- [ ] Connector health verified (no errors in HubSpot/Porter/Instantly)

## Documentation
- [ ] README-FOR-CLIENT.md customized with links and contacts
- [ ] Video walkthrough recorded (Loom/Drive link)
- [ ] Credentials handoff documented (where secrets live)
- [ ] Maintenance checklist reviewed with client

## Communication
- [ ] Final email sent with dashboard + documentation links
- [ ] SLA/retainer options presented (if applicable)
- [ ] Reminder: data may take 24-48 hours to fully populate

## Archive
- [ ] Copy client folder to long-term storage
- [ ] Create follow-up task for 30-day health check
- [ ] Close any temporary access once confirmed by client
