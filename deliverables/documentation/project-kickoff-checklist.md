# Project Kickoff Checklist

Use this list during the first client call to confirm requirements, access, and success metrics.

## Access & Credentials
- [ ] GTM Container ID provided (create if missing)
- [ ] GA4 Measurement ID provided (or stream created)
- [ ] HubSpot API/OAuth access confirmed
- [ ] LinkedIn Campaign Manager access confirmed
- [ ] Instantly login + webhook access confirmed
- [ ] Looker Studio editing access for your email

## Scope Confirmation
- [ ] Primary KPI documented (e.g., cost per lead)
- [ ] Platforms in scope (GA4, HubSpot, LinkedIn, Instantly)
- [ ] Dashboard audience defined (executive vs ops)
- [ ] Date range expectations (e.g., last 90 days)
- [ ] Refresh cadence (daily vs hourly)

## Data Quality
- [ ] Timezone alignment captured
- [ ] Currency alignment captured
- [ ] Existing tracking issues noted
- [ ] Historical backfill needs documented

## Delivery Plan
- [ ] Timeline agreed (3-5 business days)
- [ ] Revision window agreed (1 round included)
- [ ] Communication channel confirmed (email/Slack)
- [ ] Handoff package items listed (README, video, workbook)
