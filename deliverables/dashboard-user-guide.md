# Dashboard User Guide

## Navigation
- Overview page: topline KPIs (leads, spend, conversion rate)
- Platform tabs: GA4, HubSpot, LinkedIn, Instantly
- Filters: Channel, Campaign, Date Range

## Tips
- Hover over charts for tooltips and exact values
- Use "Reset filters" button to clear selections
- Export tables via ⋮ menu → Download CSV/PDF
- Bookmark the dashboard URL for quick access

## Troubleshooting
- Numbers look off? Check `config/timezone-settings.md` and reconciliation workbook
- Data missing? Confirm connectors healthy and refreshed today
