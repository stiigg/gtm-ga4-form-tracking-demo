# Enterprise Package

**Who it's for:** Multi-brand, multi-region clients needing governance and SLAs.

## Scope
- Up to 5 properties/platforms per brand
- BigQuery warehouse integration with partitioned tables
- SLA-backed monitoring and alerting
- Security review + access management documentation

## Pricing
- $1,500 setup per brand
- $500/month per brand ongoing (includes monitoring + quarterly strategy session)
- Custom dev: $120/hour for API work or bespoke connectors

## Deliverables
- Production GTM container with audit trail
- Looker Studio multi-page dashboard with brand-level and aggregate views
- Reconciliation workbook with variance notes and remediation plan
- Runbook for incidents and escalation contacts
