# Screenshot Placeholders

Screenshots are not included in this repository to keep the commit binary-free.
Use the following checklist when generating visual assets locally:

1. **Client vs Server Comparison** – Capture the three-column demo in `index-server-side.html`.
2. **GTM Preview Mode** – Show tags firing for the validated form submission.
3. **GA4 DebugView** – Highlight incoming events after enabling debug mode.
4. **Server Container Setup** – Document the server container configuration screen.
5. **Network Tab Comparison** – Contrast client-side vs server-side network calls.

Save the images in this folder using the same filenames referenced in the README:
- `client-vs-server-comparison.png`
- `gtm-preview-mode.png`
- `ga4-debugview.png`
- `server-container-setup.png`
- `network-tab-comparison.png`
